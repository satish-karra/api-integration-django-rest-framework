# API Integration with Django REST Framework

## 📌 Project Overview

This project demonstrates the development of scalable RESTful APIs using Django and Django REST Framework (DRF). It includes multiple modules such as Students, Employees, and Blogs, showcasing real-world backend architecture and API design.

---

## 🚀 Features

* CRUD operations using Django REST Framework
* Function-based views and class-based views
* Serializers and nested serializers
* Pagination, filtering, search, and ordering
* Modular app structure (students, employees, blogs)
* REST API design with proper routing and endpoints

---

## 🛠️ Tech Stack

* Python
* Django
* Django REST Framework (DRF)
* SQLite / MySQL
* Postman (for API testing)

---

## 📂 Project Structure

* `students/` → Student APIs and models
* `employees/` → Employee APIs and models
* `blogs/` → Blog and comments APIs
* `api/` → Core API configuration
* `django_rest_main/` → Project settings

---

## ▶️ How to Run

1. Clone the repository
   git clone https://github.com/satish-karra/api-integration-django-rest-framework

2. Navigate to the project
   cd api-integration-django-rest-framework

3. Install dependencies
   pip install -r requirements.txt

4. Run the server
   python manage.py runserver
